# JMIR Medical Informatics Submission Checklist

## Pre-Submission Requirements

- [x] **ORCID Required:** Get your ORCID at https://orcid.org (free, 2 minutes)
- [x] **Cover Letter:** Created (see cover_letter.md)
- [x] **Not Previously Published:** Confirmed
- [x] **File Format:** Word .docx format
- [x] **Author Guidelines:** Reviewed at https://medinform.jmir.org/author-information

## Manuscript Structure (IMRAD Format)

- [x] **Title:** Clear, descriptive, <150 characters
- [x] **Abstract:** Structured, <450 words (Background, Objective, Methods, Results, Conclusions)
- [x] **Keywords:** 5-10 keywords included
- [x] **Introduction:** Theory, hypotheses, prior work
- [x] **Methods:** Study design, recruitment, analysis, ethical considerations
- [x] **Results:** Organized with subheadings matching methods
- [x] **Discussion:** Interpretation, limitations, future work
- [x] **Conclusions:** Summary of key findings
- [x] **References:** 25-30 references, AMA style

## Required Elements

- [x] **Tables:** Maximum 2 tables (we have 2)
- [x] **Figures:** Maximum 4 figures (we have 2 + architecture diagram)
- [x] **Reporting Guideline:** N/A (not RCT, systematic review, or scoping review)
- [ ] **Data Availability Statement:** Add to manuscript
- [ ] **Ethics Statement:** Add to manuscript (computational study, no IRB needed)
- [ ] **Conflicts of Interest:** Declared in cover letter
- [ ] **Acknowledgments:** Added to manuscript

## Formatting Requirements

- [x] **Font:** Times New Roman or Arial
- [x] **Size:** 12pt
- [x] **Line Spacing:** Double-spaced
- [x] **Margins:** 1 inch all sides
- [x] **Page Numbers:** Included
- [x] **Headings:** Hierarchical (H1, H2, H3)
- [x] **References:** AMA Manual of Style, 11th Edition
- [x] **P-values:** Reported correctly (not as "p<0.05" unless <0.001)

## Supplementary Materials

- [x] **Appendix 1:** Full confusion matrices (both systems)
- [x] **Appendix 2:** Sample LLM outputs with reasoning
- [x] **Appendix 3:** Dataset distribution statistics
- [ ] **Code Repository:** Link to GitHub (https://github.com/jemiridaniel/malaria-llm-cdss)

## Author Information

- [ ] **All author names entered in metadata form** (not just in manuscript)
- [ ] **Affiliations complete**
- [ ] **Corresponding author designated**
- [ ] **ORCID linked for all authors**

## Final Checks

- [ ] **Proofread for grammar/spelling**
- [ ] **All figures high resolution (300 DPI minimum)**
- [ ] **Tables in Word format (not images)**
- [ ] **References formatted consistently**
- [ ] **Supplementary files named correctly**
- [ ] **Cover letter uploaded**
- [ ] **Manuscript uploaded as .docx**

## Submission Steps

1. Create account at https://medinform.jmir.org
2. Click "Make a New Submission"
3. Upload cover letter
4. Upload manuscript (.docx)
5. Upload figures (separate files)
6. Upload supplementary materials
7. Enter all metadata (authors, keywords, abstract)
8. Review and submit

## Post-Submission

- [ ] **Confirmation email received**
- [ ] **Manuscript number noted**
- [ ] **Editor assigned** (wait 1-2 weeks)
- [ ] **Peer review** (wait 6-10 weeks)

## Estimated Timeline

- Submission: Today (Dec 3, 2025)
- Editor decision: ~2 weeks
- Peer review: ~8-10 weeks
- Revision: ~2 weeks
- Final decision: ~12-14 weeks
- Publication: ~2 weeks after acceptance

**Target publication date: March 2025**
